# Vortex E-commerce Site

Welcome to the Vortex E-commerce Site project! This project is designed to provide a seamless experience for users looking to create and manage their own websites through our subscription plans.

## Features

- User-friendly interface for website creation
- Multiple subscription plans to choose from
- Secure payment processing
- Customer testimonials and feedback
- Responsive design for all devices

## Installation

To get started with the Vortex E-commerce Site, clone the repository and install the dependencies:

```bash
git clone <repository-url>
cd vortex-ecommerce-site
npm install
```

## Usage

To run the development server, use the following command:

```bash
npm run dev
```

You can then access the site at `http://localhost:3000`.

## Components

The project consists of several key components:

- **Header**: Navigation bar for the site.
- **Hero**: Main call-to-action section on the homepage.
- **Plans**: Displays available subscription plans.
- **PricingCard**: Individual pricing options within the Plans component.
- **Features**: Highlights key features of the service.
- **Testimonials**: Displays customer reviews.
- **Footer**: Contains footer information.
- **Modal**: Handles modal dialogs for interactions.
